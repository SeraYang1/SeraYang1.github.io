Alexa Nutrition Tracker: Amazon Alexa application that helps the user keep track of what they consumed. The user can ask Nutrition Tracker to find the caloric infromation on a food or record foods that the user consumed each day. Then, based on the user's entered personal information (such as height and weight), Nutrition Tracker recommends foods based on how many calories the user should be consuming and how many they have already consumed in one day.

Hangman: The word guessing game, Hangman, created from scratch. The user is given how many letters are in a word, then has to guess letters to fill in the blanks until they can guess the word. For each incorrect guess, another part of the hangman is drawn. Animations and UI are designed on the JFrame. User inputs are read in through ActionListener and KeyListener. Words are randomly choosen from a text file of dictionary words.
